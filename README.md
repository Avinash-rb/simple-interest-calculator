#  Simple Interest Calculator

A straightforward tool to calculate **simple interest** based on principal amount, annual interest rate, and time.

---

##  Features

- Calculate simple interest instantly  
- Easy-to-use interface  
- Accurate and fast computation  
- Beginner-friendly project  

---

##  Formula Used
I = (P × R × T) / 100

### Where:

- **I** → Total Interest  
- **P** → Principal Amount  
- **R** → Annual Interest Rate (%)  
- **T** → Time (in years)
